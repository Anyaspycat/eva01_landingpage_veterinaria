# 🐱 Landing Page Veterinaria Anya

## Descripción del Proyecto

Este proyecto consiste en el desarrollo de una **Landing Page para la Veterinaria Anya**, especializada en la atención y cuidado de gatos. El sitio web fue diseñado con el objetivo de proporcionar información clara sobre los servicios ofrecidos, facilitar el contacto con potenciales clientes y generar confianza mediante la presentación de tratamientos y reseñas de clientes.

---

## Problemática Abordada

La **Veterinaria Anya** requiere una página tipo landing page orientada a captar nuevos clientes que buscan atención profesional y cuidados especializados para sus felinos.

La solución desarrollada permite:

* Presentar los servicios y tratamientos disponibles.
* Facilitar el contacto mediante un formulario de atención.
* Mostrar testimonios y reseñas de clientes.
* Mejorar la presencia digital de la veterinaria.
* Entregar una experiencia visual atractiva y responsiva.

---

## Tecnologías Utilizadas

Las tecnologías empleadas para el desarrollo del proyecto fueron:

* **HTML5** para la estructura del sitio web.
* **CSS3** para los estilos personalizados.
* **Bootstrap 5** como framework para el diseño responsivo.
* **Git** para el control de versiones.
* **GitHub** como repositorio remoto del proyecto.
* **GitHub Copilot** como herramienta de Inteligencia Artificial para asistencia en el desarrollo.
* **Vercel** para la publicación y despliegue del sitio.
